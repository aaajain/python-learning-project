=============== Steps to Freeze dependency(requirement) ==================

1. Open CMD and go till the project path, then type (Before pushing code on github)
		command) pip freeze > requirements.txt
					- This will create requirements text file where all the dependencies name will get created.(you can give any name to file)
					
		command) pip install -r requirements.txt
					- It will install the defined dependencies in requirement file.(After checking out code with requirement file)
